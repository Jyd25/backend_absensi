# ============================================
# VPS Deployment Guide
# Backend Sistem Kehadiran — applab.my.id
# ============================================

## Overview

- **Domain**: applab.my.id
- **Backend**: Laravel 12 (PHP 8.2 + Nginx + Supervisor)
- **Database**: Neon PostgreSQL (cloud)
- **Frontend**: Vercel (https://frontend-jyd25.vercel.app)
- **Stack**: Ubuntu 22.04/24.04, PHP 8.2-FPM, Nginx, Supervisor, Redis

## Prerequisites

- VPS dari Rumahweb (Ubuntu 22.04/24.04)
- Domain applab.my.id sudah pointing ke IP VPS (A record)
- SSH access ke VPS

## Quick Start

### 1. Point Domain

Di DNS provider (Rumahweb), tambahkan:

| Type | Name | Value | TTL |
|------|------|-------|-----|
| A | @ | `IP_VPS_ANDA` | 300 |
| A | www | `IP_VPS_ANDA` | 300 |

### 2. Initial VPS Setup

```bash
# SSH ke VPS
ssh root@IP_VPS_ANDA

# Upload & jalankan setup script
scp deploy/setup-vps.sh root@IP_VPS_ANDA:/tmp/
ssh root@IP_VPS_ANDA "bash /tmp/setup-vps.sh"
```

### 3. Clone & Configure Backend

```bash
# Switch to deploy user
su - deploy

# Clone backend
cd /var/www/sistem-kehadiran
sudo -u deploy git clone https://github.com/Jyd25/backend_absensi.git .
sudo -u deploy git checkout main

# Install dependencies
composer install --no-dev --optimize-autoloader

# Setup .env
cp .env.example .env
php artisan key:generate
nano .env  # Edit environment variables
```

### 5. Run Migrations & Seed

```bash
php artisan migrate --force
php artisan db:seed --force  # Optional: seed awal
```

### 6. Setup Nginx

```bash
# Copy nginx config
sudo cp deploy/nginx-applab.conf /etc/nginx/sites-available/sistem-kehadiran
sudo ln -sf /etc/nginx/sites-available/sistem-kehadiran /etc/nginx/sites-enabled/
sudo rm -f /etc/nginx/sites-enabled/default

# Test & reload
sudo nginx -t
sudo systemctl reload nginx
```

### 7. Setup SSL (HTTPS)

```bash
# Pastikan domain sudah pointing
sudo certbot --nginx -d applab.my.id -d www.applab.my.id
```

Setelah SSL, uncomment block HTTPS di nginx config:
```bash
sudo nano /etc/nginx/sites-available/sistem-kehadiran
# Uncomment the HTTPS server block
sudo nginx -t && sudo systemctl reload nginx
```

### 8. Setup Supervisor (Queue + WebSocket)

```bash
# Copy configs
sudo cp deploy/supervisor-worker.conf /etc/supervisor/conf.d/
sudo cp deploy/supervisor-reverb.conf /etc/supervisor/conf.d/

# Reload supervisor
sudo supervisorctl reread
sudo supervisorctl update
sudo supervisorctl start "sistem-kehadiran-worker:*"
sudo supervisorctl start sistem-kehadiran-reverb

# Verify
sudo supervisorctl status
```

### 9. Setup Cron (Scheduler)

```bash
# Add to crontab
(crontab -l 2>/dev/null; echo "* * * * * cd /var/www/sistem-kehadiran && php artisan schedule:run >> /dev/null 2>&1") | crontab -
```

### 10. Cache Optimization

```bash
cd /var/www/sistem-kehadiran
php artisan config:cache
php artisan route:cache
php artisan view:cache
php artisan event:cache
```

## Deployment Commands

### Update After Git Push

```bash
cd /var/www/sistem-kehadiran
bash deploy/deploy.sh
```

### Manual Service Management

```bash
# Queue worker
sudo supervisorctl restart sistem-kehadiran-worker:*

# Reverb WebSocket
sudo supervisorctl restart sistem-kehadiran-reverb

# Nginx
sudo systemctl reload nginx

# PHP-FPM
sudo systemctl reload php8.2-fpm

# Check logs
tail -f /var/www/sistem-kehadiran/storage/logs/worker.log
tail -f /var/www/sistem-kehadiran/storage/logs/reverb.log
```

### SSL Renewal

```bash
sudo certbot renew --dry-run
```

## Troubleshooting

### 502 Bad Gateway
```bash
# Check PHP-FPM
sudo systemctl status php8.2-fpm
sudo systemctl restart php8.2-fpm

# Check socket
ls -la /var/run/php/php8.2-fpm.sock
```

### Queue Worker Not Running
```bash
sudo supervisorctl status
sudo supervisorctl restart sistem-kehadiran-worker:*
cat /var/www/sistem-kehadiran/storage/logs/worker.log
```

### Reverb WebSocket Not Connecting
```bash
sudo supervisorctl status sistem-kehadiran-reverb
cat /var/www/sistem-kehadiran/storage/logs/reverb.log

# Check port 8080
sudo netstat -tlnp | grep 8080
```

### Permission Issues
```bash
sudo chown -R deploy:www-data /var/www/sistem-kehadiran
sudo chmod -R 775 /var/www/sistem-kehadiran/storage
sudo chmod -R 775 /var/www/sistem-kehadiran/bootstrap/cache
```

### Database Connection
```bash
php artisan tinker
>>> DB::connection()->getPdo();  # Should return PDO object
```
